## 📝 Descripción del Proyecto
GameVault es una aplicación móvil desarrollada en React Native que funciona como un catálogo interactivo de videojuegos. Permite a los usuarios explorar una lista de títulos, consultar detalles específicos con navegación fluida, leer noticias del mundo gaming que rotan automáticamente y simular la adición de nuevos juegos mediante un formulario.

## 💻 Tecnologías Utilizadas
Framework:** React Native CLI (Funcional con Hooks)
Navegación:** React Navigation (Stack Navigation & Bottom Tabs)
Gestión de Estado y Ciclo de Vida:** React Hooks (`useState`, `useEffect`)
Interfaz de Usuario:** React Native Safe Area Context, Vector Icons (Ionicons)
Plataforma de pruebas:** macOS / iOS Simulator
